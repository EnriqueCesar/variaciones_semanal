Variación Semanal 5.0

Proyecto PWA listo para GitHub Pages.

Mejoras aplicadas:
- Filtros simples e intuitivos: año, mes, semana, DM, tienda, categoría e ingrediente con lista desplegable.
- Selector de vista principal: $ o unidades.
- Gráficas especiales de variación $ y variación en unidades: faltante arriba, sobrante abajo y neto como línea.
- Top faltantes y top sobrantes con botón de $ / unidades y selector rápido de Top 5, 10, 15, 20, 25.
- KPIs más claros: impacto neto, faltante $, sobrante $, producto foco y score salud.
- Foco recomendado con lectura ejecutiva para actuar rápido.
- Excel actualizado en data/Variacion_Semanal.xlsx.

Publicación:
1. Sube todos los archivos a la raíz del repositorio.
2. En GitHub Pages selecciona Branch main / root.
3. Abre el sitio publicado.
